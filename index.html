Capabilities

What It Does

🔍

Web Research

Searches the web via Brave Search API to find the latest news, trends, and data on any topic before writing.

🤖

AI Drafting

Claude analyzes research and writes engaging, informed tweets — under 280 characters, sounding human, not robotic.

🖼️

Image Generation

Runware AI creates eye-catching visuals that get attached directly to your tweets. Claude decides when images help.

✅

One-Tap Approval

Review the draft, tap approve, and it's live on Twitter/X. Full control, zero friction.

Open Source

Luffa Python SDK

We built and published the first community SDK for the Luffa Bot API. Telegram-style DX, published on PyPI.

$ pip install luffa-sdk
from luffa import LuffaBot, button

bot = LuffaBot(secret="your-secret")

@bot.command("hello")
async def hello(msg):
    await msg.reply("Hey there!")

bot.run()
